Visión de conjunto

Para mostrar cualquier información en su tablero, debe enviar los cambios de información de su dispositivo a nuestra API. Estas apis, todas requieren un token para distinguir la identidad.

Para mayor flexibilidad, podemos aceptar un token por dispositivo diferente o un token general que puede describir información para todos los dispositivos, y en el cuerpo de la solicitud, puede especificar el dispositivo por su id. En general, nuestros conjuntos de aplicaciones se utilizan para recopilar información, pero también puede encontrar algunas aplicaciones que le devuelven información útil sobre su ecosistema.

Es muy importante notarTodas las respuestas de su API del servidor vuelven con una guía de Google, Llamado guía de estilo de google jsony puede adivinar el éxito y el fracaso de su solicitud de API según este formato. En términos generales, si nuestros servidores pueden captar su solicitud, devolveremos 200 respuestas de todos modos, incluso si la operación no ha realizado. En este caso, debe buscar el campo de error dentro del objeto de respuesta.

Las respuestas que tienen 50x, 40x o más, rara vez aparecen porque nuestro objetivo es manejar todas las situaciones posibles con nuestra API. y debe considerar este tipo de errores temporalmente fuera de servicio o tiempo de inactividad parcial.
